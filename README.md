##  我做的改变
### 1. 使用vite创建项目而不是CRA
vite比CRA创建跟快，重新构建速度也更快，所以vite是比CRA跟好的选择。使用vite创建React(JavaScript)项目的command（我喜欢使用bun）：
```
# 我事先把文件夹创建好了，在文件夹内调用的vite，所以传入 . 而不是项目的名字
$ bun create vite . --template react
```
